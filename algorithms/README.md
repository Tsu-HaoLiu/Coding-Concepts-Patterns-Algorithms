## Two Pointers 

Check if string is Palindrome, using two pointers.
```python
def two_pointers(array):

    start = 0
    end = len(array) - 1

    while start != end:

        if array[start] != array[end]:
            return False

        start += 1
        end -= 1
    return True
```
